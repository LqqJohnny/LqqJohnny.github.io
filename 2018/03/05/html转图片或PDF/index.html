今天带来一个好玩的网页截图功能 ，就是将网页转换成一张图片 或者 PDF

刚开始发现网上有很多 方案 ，例如 ： http://blog.csdn.net/younglao/article/details/77746039

这里介绍了三种：

  • html-pdf 模块
  • wkhtmltopdf 工具
  • phantom 模块

这三种方法各有优缺点，视情况选择。

其中我用过 phantom ，所以我就试着用它来实现该功能 。
